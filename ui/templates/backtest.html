How It Works
  1. Enter the Forecast ID from a previously created forecast
  2. Enter the Realised Prices that actually occurred during the forecast period (comma-separated)
  3. Click "Run Backtest Evaluation" to compare the forecast against actual outcomes
  4. Review the evaluation metrics:
    • MAE (Mean Absolute Error): Average absolute difference between predicted and actual prices
    • MAPE (Mean Absolute Percentage Error): Average percentage error
    • RMSE (Root Mean Squared Error): Square root of average squared errors
    • Directional Accuracy: Percentage of correctly predicted price direction (up/down)
    • Calibration: Whether the forecast uncertainty bounds match actual outcomes

Tip: You can get a Forecast ID from the Forecast Dashboard after creating a forecast. Make sure the number of realised prices matches the forecast horizon.
